# Media pipeline

Place approved source media outside the public route, then export web-ready assets here.

- Use AVIF or WebP for photographs, SVG for simple marks, and MP4/WebM for intentional motion.
- Store at least two responsive widths for editorial images (roughly 960px and 1920px).
- Keep meaningful alt text in the content record, not in the filename.
- Do not publish an entire full-resolution event gallery. Curate the edit and lazy-load below the fold.
- Record photographer credit and usage approval before adding an archive gallery entry.
